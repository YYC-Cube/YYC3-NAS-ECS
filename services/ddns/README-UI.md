# YYC3 智能应用平台的UI/UX系统设计

> 设计一个基于FRP智能应用平台的UI/UX系统。拒绝黑色及近黑色，需要选择一个明亮或柔和的色彩方案。

 设计目标：

 1. 全页面设计：包括登录页、控制台首页、FRP服务管理、监控大屏、知识库、工作流设计器、系统设置等。
 2. 组件库：设计一套统一的组件，包括按钮、输入框、表格、卡片、导航栏、侧边栏、图表等。
 3. 响应交互：确保在不同设备上的适配，以及交互动效（如悬停、点击、加载）。
 4. 视觉阴影：使用柔和的阴影来营造层次感。
 5. 色彩区分：使用一套非黑色系的色彩方案，区分不同功能、状态。

 色彩方案建议（示例）：
  主色：蓝色系（如#4A6FA5）
  辅助色：绿色系（如#4CAF50）、橙色系（如#FF9800）、紫色系（如#9C27B0）
  背景色：浅灰色（如#F5F5F7）、白色（如#FFFFFF）
  文字颜色：深灰色（如#333333、#666666）
  错误色：红色系（如#F44336）
  成功色：绿色系（如#4CAF50）
  警告色：橙色系（如#FF9800）

 阴影方案：使用浅阴影，例如：box-shadow: 0 2px 12px 0 rgba(0,0,0,0.1);

 由于这是一个全量提示词，我们需要描述每个页面的布局、组件和交互。但是，由于篇幅限制，我们只能给出一个概述，并详细设计几个关键页面和组件。

 我们选择设计以下页面：

  1. 登录页
  2. 控制台首页（仪表板）
  3. FRP服务管理页面
  4. 监控大屏
  5. 知识库页面
  6. 工作流设计器

 我们为每个页面提供设计提示词，包括布局、使用的组件、交互和色彩。

 同时，我们设计一套组件库，包括按钮、输入框、表格、卡片等。

 最后，我们提供响应式设计和交互动效的指导。

 由于这是一个文本提示词，我们将使用描述性语言，并给出一些关键的CSS样式示例。

```css
6. 内阴影（输入框聚焦）：
   box-shadow: inset 0 2px 4px 0 rgba(0, 0, 0, 0.05);

7. 彩色阴影（强调元素）：
   .primary-shadow {
     box-shadow: 0 4px 14px 0 rgba(30, 58, 138, 0.2);
   }
   
   .success-shadow {
     box-shadow: 0 4px 14px 0 rgba(16, 185, 129, 0.2);
   }

8. 玻璃态阴影（毛玻璃效果）：
   background: rgba(255, 255, 255, 0.7);
   backdrop-filter: blur(10px);
   box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.1);
```

### **阴影应用场景**

```css
/* 阴影应用提示词 */
请在不同组件中应用合适的阴影：

1. 按钮阴影：
   - 默认按钮：无阴影或最浅阴影
   - 悬停按钮：浅阴影，增加 elevation
   - 按下按钮：中等阴影，向内的感觉
   - 浮动按钮：最深阴影，悬浮感强烈

2. 卡片阴影：
   - 静态卡片：浅阴影
   - 可交互卡片：中等阴影，悬停时加深
   - 活动卡片：深阴影，突出选中状态

3. 导航阴影：
   - 顶部导航：最深阴影，建立层次
   - 侧边栏：中等阴影，区分内容
   - 底部导航：中等阴影，轻微提升

4. 模态框阴影：
   - 背景遮罩：背景色 rgba(0,0,0,0.4)
   - 模态框：深阴影，明显突出
   - 下拉菜单：深阴影，快速弹出感

5. 输入框阴影：
   - 默认状态：无阴影或内阴影
   - 聚焦状态：彩色阴影（主色系）
   - 错误状态：彩色阴影（错误色系）
```

## 🎯 视觉设计原则

### **设计系统原则**

```css
/* 设计原则提示词 */
请遵循以下视觉设计原则：

1. 一致性原则：

   - 组件样式保持一致
   - 动画时长和缓动函数统一
   - 间距和尺寸使用统一规范

2. 层次性原则：
   - 使用阴影建立视觉层次
   - 重要内容使用更大字号或更亮颜色
   - 相关信息分组显示
   - 操作流程有明确视觉引导

3. 反馈性原则：
   - 所有操作都有视觉或动效反馈
   - 状态变化有明显视觉指示
   - 加载和等待状态明确显示
   - 错误和成功有明确提示

4. 效率性原则：
   - 常用操作路径最短
   - 提供快捷键和快捷方式
   - 智能预测和自动完成
   - 批量操作支持

5. 容错性原则：
   - 危险操作有确认提示
   - 提供撤销和重做功能
   - 表单错误明确提示和纠正建议
   - 网络错误有重试机制

   - 足够的颜色对比度（至少4.5:1）
   - 支持键盘导航
   - 屏幕阅读器友好
   - 不依赖颜色传达信息
   - 文字可调整大小

7. 美观性原则：
   - 使用和谐的色彩搭配
   - 适当的留白和间距

   - 优雅的动画过渡

- 间距系统：4px为基数的倍数（4、8、12、16、20、24...）
- 字体层次：h1-6、标题、正文、小字、极小字
- 图标风格：线性、面性、双色，统一视觉重量
- 图片处理：统一圆角、阴影、悬停效果
```
