# ⚡ # YYC³ （YanYuCloudCube）快速启动邮箱系统

## 最简单的方式（1分钟搞定）

### 步骤1：修改App.tsx

用以下内容**完全替换** `/src/app/App.tsx` 文件：

```tsx
import { EmailApp } from './EmailApp';

export default EmailApp;
```

就这么简单！🎉

### 步骤2：运行应用

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
```

打开浏览器访问开发服务器地址（通常是 <http://localhost:5173），你就能看到完整的企业邮箱系统了！>

### 修改主题颜色

编辑 `/src/styles/theme.css` 第43-76行，修改CSS变量：

```css
/* CPU模块（蓝色）- 默认主题 */
--module-cpu-primary: #2A6EBB;  /* 改成你喜欢的主色 */
--module-cpu-shadow: #1E5CA3;   /* 阴影色 */
--module-cpu-dark: #0D3A6A;     /* 深色文本 */
--module-cpu-light: #F0F7FF;    /* 浅背景 */
```

## ❓ 常见问题

### Q: 修改App.tsx后页面空白？

A: 检查浏览器控制台是否有错误，确认Motion包已正确安装

### Q: 主题切换按钮点击无效？

A: 清除浏览器缓存，刷新页面重试

### Q: 邮件列表显示"加载中"不消失？

A: 检查浏览器控制台Network标签，确认没有网络错误

### Q: 想恢复原来的应用？

A: 备份你的原始App.tsx，需要恢复时替换回去即可
